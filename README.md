# SENITO Healthcare

## Overview

SENITO is a full-stack healthcare application that connects patients and caregivers through a secure and user-friendly platform.

## Features

- Secure Login
- Role-Based Authentication
- Patient Dashboard
- Caregiver Dashboard
- Health Monitoring
- Patient-Caregiver Communication

## Tech Stack

- Java
- React.js
- Node.js
- SQLite

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

Run backend

Run frontend
